# Weather Forecasting Site

A website that shows weather forecasts for any city.

## Technologies
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## Features
- Search weather by city
- Displays temperature, humidity, and weather conditions
- Responsive design for mobile and desktop
